About Flow Timer

Creator

Ilya Fisiuk

Flow Timer was created by Ilya Fisiuk.

What is Flow Timer?

Flow Timer is an ADHD-friendly pomodoro timer application that recognizes and supports flow states. Unlike traditional pomodoro timers that interrupt you when you're in the zone, Flow Timer detects when you're in a flow state and allows you to continue working without disruption.

Features

  • Traditional pomodoro functionality with customizable focus and break periods
  • Flow state detection and support
  • Visual cues to indicate your current state
  • Dark mode support
  • Analytics to track your productivity
  • Responsive design that works on all devices

Version History

  • v1.3.1 - Improved button functionality and UI with better responsive layout
  • v1.3.0 - Fixed UI issues with chart sizing and added visual cue preview
  • v1.2.0 - Major rewrite for better reliability and button functionality
  • v1.1.0 - Restructured timer logic into separate module
  • v1.0.0 - Initial release
